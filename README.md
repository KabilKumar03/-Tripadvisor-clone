 TripAdvisor Clone

This project is a responsive TripAdvisor-inspired web interface built using HTML and CSS.

 Features
- Responsive layout using Flexbox
- Image cards with text overlay
- Sticky navigation bar
- Hover effects for better user experience

 Tech Stack
- HTML5
- CSS3
